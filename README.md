# Hot Sales Push

Backend de Web Push do Hot Sales para deploy na Vercel.

## Variáveis

Copie `.env.example` e configure no painel da Vercel:

- `VAPID_SUBJECT`
- `VAPID_PUBLIC_KEY`
- `VAPID_PRIVATE_KEY`
- `PUSH_API_SECRET`
- `ALLOWED_ORIGINS`
- `UPSTASH_REDIS_REST_URL`
- `UPSTASH_REDIS_REST_TOKEN`

## Deploy

Importe este repositório na Vercel.
